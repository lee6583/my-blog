# Vue 页面性能排查手册

这篇 demo 用于展示首页文章卡片的字段映射效果。

## 三步排查法

- 先看网络瀑布图，确认是否资源阻塞。
- 再看主线程任务，定位长任务来源。
- 最后看组件级渲染次数，避免重复更新。

## 优先优化项

- 拆分首屏 JS。
- 延迟非关键模块加载。
- 减少首屏同步计算。
